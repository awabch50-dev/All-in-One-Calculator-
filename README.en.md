## 🚀 Quick Start

This is a pure front-end web application, which means it doesn't require any complex server setup or databases.

<details>
<summary><b>Click to see installation steps</b></summary>
<br>

1. **Download or clone** the project.
2. **Serve the project folder** using any local server.

For example, if you have Python installed:
```bash
# Navigate to the project folder
cd "ALL IN ONE CALCULATOR AWAB PR"

# Start Python's built-in server
python -m http.server 3456
```
Then, open `http://127.0.0.1:3456/` in your web browser.
</details>

## 💻 Technologies Used

- <img src="https://cdn.jsdelivr.net/gh/devicons/devicon/icons/html5/html5-original.svg" width="20" align="top"> **HTML5**: For semantic structure.
- <img src="https://cdn.jsdelivr.net/gh/devicons/devicon/icons/css3/css3-original.svg" width="20" align="top"> **CSS3**: For stunning animations, CSS Variables, glassmorphism, and responsive design.
- <img src="https://cdn.jsdelivr.net/gh/devicons/devicon/icons/javascript/javascript-original.svg" width="20" align="top"> **Vanilla JavaScript (ES6+)**: For logic, DOM manipulation, and data fetching (no heavy frameworks).
- 🔌 **APIs**: `fawazahmed0/currency-api` for live and historical currency rates.
- 📅 **Libraries**: [Flatpickr](https://flatpickr.js.org/) for date selection in the Age Calculator.
