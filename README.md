## 🚀 فوری آغاز

یہ ایک خالصتاً فرنٹ اینڈ (Front-end) ویب ایپلیکیشن ہے جس کے لیے کسی پیچیدہ سرور یا ڈیٹابیس کی ضرورت نہیں ہے۔ 

<details>
<summary><b>پروجیکٹ کو چلانے کا طریقہ دیکھنے کے لیے کلک کریں</b></summary>
<br>

1. **پروجیکٹ کو ڈاؤن لوڈ** یا کلون کریں۔
2. **کسی بھی مقامی سرور** (Local Server) کے ذریعے اس پروجیکٹ فولڈر کو پیش (serve) کریں۔

مثال کے طور پر، اگر آپ کے پاس Python موجود ہے:
```bash
# پروجیکٹ کے فولڈر میں جائیں
cd "ALL IN ONE CALCULATOR AWAB PR"

# Python کا بلٹ ان سرور چلائیں
python -m http.server 3456
```
اس کے بعد اپنے ویب براؤزر میں `http://127.0.0.1:3456/` کھولیں۔
</details>

## 💻 ٹیکنالوجیز (Technologies Used)

- <img src="https://cdn.jsdelivr.net/gh/devicons/devicon/icons/html5/html5-original.svg" width="20" align="top"> **HTML5**: سیمنٹک (Semantic) ساخت کے لیے۔
- <img src="https://cdn.jsdelivr.net/gh/devicons/devicon/icons/css3/css3-original.svg" width="20" align="top"> **CSS3**: شاندار اینیمیشنز، CSS Variables، گلاس مارفزم، اور ریسپانسو (Responsive) ڈیزائن کے لیے۔
- <img src="https://cdn.jsdelivr.net/gh/devicons/devicon/icons/javascript/javascript-original.svg" width="20" align="top"> **Vanilla JavaScript**: لاجک، DOM مینیپولیشن اور APIs سے ڈیٹا حاصل کرنے کے لیے۔
- 🔌 **APIs**: `fawazahmed0/currency-api` لائیو اور تاریخی کرنسی ریٹس کے لیے۔
- 📅 **لائبریریز**: عمر کیلکولیٹر میں تاریخ کے انتخاب کے لیے [Flatpickr](https://flatpickr.js.org/) کا استعمال۔
